## 快速排序算法简介

快速排序（Quick Sort）是一种高效的分治排序算法，由 Tony Hoare 在 1960 年提出。

### 算法特点

- **时间复杂度**:
  - 平均情况: O(n log n)
  - 最坏情况: O(n²) - 当数组已经有序且选择第一个/最后一个元素作为基准时
  - 最好情况: O(n log n)
  
- **空间复杂度**: O(log n) - 用于递归调用栈

- **稳定性**: 不稳定排序

### 算法原理

1. **选择基准（Pivot）**: 从数组中选择一个元素作为基准
2. **分区（Partition）**: 将数组分为两部分，小于基准的元素和大于基准的元素
3. **递归排序**: 对两部分递归应用快速排序
4. **合并**: 将排序后的子数组合并

## 功能实现

本项目提供了两种快速排序实现：

### 1. 函数式快速排序 (`quick_sort`)

```python
quick_sort(arr)
```

- 创建新列表，不修改原数组
- 代码简洁易懂
- 使用列表推导式进行分区

### 2. 原地快速排序 (`quick_sort_inplace`)

```python
quick_sort_inplace(arr, low=0, high=None)
```

- 直接在原数组上操作
- 空间复杂度更低
- 适合处理大型数据集

## 使用方法

### 基本使用

```python
from sort-test import quick_sort

# 待排序数组
arr = [64, 34, 25, 12, 22, 11, 90]

# 调用快速排序
sorted_arr = quick_sort(arr)

print(f"原始数组: {arr}")
print(f"排序结果: {sorted_arr}")
```

### 原地排序

```python
from sort-test import quick_sort_inplace

# 待排序数组
arr = [64, 34, 25, 12, 22, 11, 90]

# 原地排序（直接修改原数组）
quick_sort_inplace(arr)

print(f"排序结果: {arr}")
```

## 优化说明

本项目实现包含了以下优化：

1. **基准选择优化**: 选择中间元素作为基准，避免最坏情况
2. **重复元素处理**: 将等于基准的元素单独处理，减少递归次数
3. **原地排序选项**: 提供空间复杂度更低的原地排序版本
