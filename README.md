## 📂 项目列表

### 🐍 贪吃蛇游戏
- **目录**：[snake/](./snake)
- **在线访问**：[贪吃蛇游戏](https://andrew8yang.github.io/opencodetest/)
- **技术栈**：HTML5, CSS3, JavaScript, Canvas API
- **功能**：
  - 经典贪吃蛇玩法
  - 响应式设计
  - 最高分记录
  - 暂停功能

## 🚀 如何使用

### 查看特定项目
```bash
cd snake
```

### 在线运行
直接访问项目的 GitHub Pages 链接即可在线运行。

### 本地运行
```bash
# 进入项目目录
cd snake

# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```
